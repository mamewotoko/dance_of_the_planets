# Dance of the planets (BuckleScript)

## setup

1. 

    ```
    sh setup.sh
    ```
    
## build
1.

    ```
    sh build.sh
    ```

   A file `js/dance.js` will be generated.

## view result
1. Start web server. For example, 

    ```
    python3 -m http.server
    ```

2. browse <http://localhost:8000/>

* bsconfig.json
* package.json
* src/*.ml
