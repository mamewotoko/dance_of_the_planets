# Dance of the planets (BuckleScript)

![earth venus](img/earth_venus.png)

## Setup

1. 

        sh setup.sh
    
## Build
1.

        sh build.sh

   A file `js/dance.js` will be generated.

## View result
1. Start web server. For example, 

        python3 -m http.server

2. browse <http://localhost:8000/>

## Source files

* bsconfig.json
* package.json
* src/*.ml
