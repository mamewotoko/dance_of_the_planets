Kotlin/JS
================

Build
-----

```
./gradlew build
```


Run
-------

```
./gradlew browserRun
```

A Web browser will start.

Or browse [./build/distributions/index.html](./build/distributions/index.html)

Files
---------

```
./src/main/
       src/main.kt
       resources/index.html
```
