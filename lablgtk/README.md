Ocaml lablgtk: Dance of the planets
=====================================

Setup
--------
1. install gtk3

    ```bash
    brew install gtk
    ```
2. install opam2
   <https://opam.ocaml.org/doc/Install.html>
3. install lablgtk

    ```
    sh setup-opam
    ```

Build
------

1.

    ```
    make
    ```
    
Run
----

    ```
    ./dance
    ```
