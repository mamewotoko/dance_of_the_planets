# Dance of the planets (OCaml)

A sample program using Graphics to draw lines, text.

![Earth Venus](img/earth_venus_ocaml.png)

## Install

1. 

        opam install odoc graphics dune

## Build

1. 

        dune build src/dance.exe

## Run

1.

        dune exec src/dance.exe
