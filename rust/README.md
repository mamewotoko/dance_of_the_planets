Rust: The dance of the Planets
=================================

Method1: cargo installed environment
------------------------------------

### Build


```
cargo build
```

### Run

```
cargo run
```

Method2: Vagrant
-----------------

### Build

```
vagrant up
```

Method3: docker-compose
------------------------

### Build

```
docker-compose build
docker-compose run rust
```
