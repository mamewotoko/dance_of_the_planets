# Dance of the planets (ScalaJS)

## Setup

1. [Install sbt](https://www.scala-sbt.org/1.x/docs/Setup.html)
    * Mac

        brew install sbt


## Build

1. 

        sbt fastOptJS

Open `index-dev.html` with web browser.
